SmartBot Assistant is a lightweight AI-powered chatbot for WordPress that answers user questions, suggests content, and learns from real conversations. Supports OpenAI, Claude, Gemini, and OpenRouter.

== Description ==

SmartBot Assistant is an AI chatbot and content helper for WordPress websites. It enhances user engagement, surfaces helpful content, and supports multi-model AI responses.

**Key Features:**
* Custom FAQ builder with keyword-based matching
* Dynamic AI fallback with OpenAI, Claude, Gemini, or OpenRouter
* Shortcode and floating chatbot support
* Logs and query review dashboard
* Create new FAQs from AI-generated responses
* Developer-friendly architecture and extendable logic

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/smartbot-assistant` directory.
2. Activate it through the ‘Plugins’ menu in WordPress.
3. Go to **SmartBot → Settings** to configure AI provider and API keys.
4. Use `[smartbot_chat]` shortcode or rely on the default floating UI.

== Frequently Asked Questions ==

= Is it free to use? =
Yes, this version includes core features and supports your own API keys for major AI providers.

= Which models are supported? =
You can use OpenAI (GPT-3.5, GPT-4), Claude 3, Gemini Pro, or any OpenRouter model.

= Can I log conversations? =
Yes. You can view queries, AI responses, and create FAQs directly from the logs.
